# Definition of Done (LUVI)

- ✅ CI grün: `flutter analyze` & `flutter test`
- ✅ Tests vorhanden: Unit/Widget passend zur Story
- ✅ MIWF befolgt (Happy Path zuerst, keine vorauseilenden Guards)
- ✅ DSGVO-Review-Dokument aktualisiert (docs/privacy/reviews/<branch>.md)
- ✅ PR-Template ausgefüllt (Summary/Type/Testing/MIWF/DoD)
- ✅ ADRs aktualisiert, falls Architektur-/Security-Entscheidung betroffen
