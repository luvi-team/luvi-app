# Chart Accessibility Checklist (A11y)

Kontrast & Lesbarkeit
- Text ≥ 4.5:1; grafische Elemente ≥ 3:1; Mindest‑Fontgrößen einhalten.

Keine Farbe‑Only Kodierung
- Alternativen: Muster, Marker‑Formen, direkte Labels/Annotations.

Text‑Alternativen
- Kurze Zusammenfassung/Alt‑Text je Chart; optional Daten‑Tabelle bereitstellen.

Tastatur & Fokus
- Filter/Interaktion per Keyboard nutzbar; sichtbarer Fokus; Screenreader‑Labels.

Tests
- Color‑Blind‑Simulator; VoiceOver/TalkBack‑Probe; High‑Contrast‑Modus; Zoom/Scaling.
