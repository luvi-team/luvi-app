# Third-Party Fonts

Diese App bündelt folgende Schriften. Alle werden unter der **SIL Open Font License 1.1** verteilt (unverändert). Lizenztext: `LICENSES/fonts/OFL-1.1.txt`.

- **Playfair Display**  
  Lizenz: SIL Open Font License 1.1  
  Lizenzdatei im Repo: `LICENSES/fonts/PlayfairDisplay-LICENSE.txt` → verweist auf `OFL-1.1.txt`  
  Verwendung: UI Headlines / Branding (siehe `pubspec.yaml`)

- **Figtree**  
  Lizenz: SIL Open Font License 1.1  
  Lizenzdatei im Repo: `LICENSES/fonts/Figtree-LICENSE.txt` → verweist auf `OFL-1.1.txt`  
  Verwendung: UI Body/Text (siehe `pubspec.yaml`)

- **Inter**  
  Lizenz: SIL Open Font License 1.1  
  Lizenzdatei im Repo: `LICENSES/fonts/Inter-LICENSE.txt` → verweist auf `OFL-1.1.txt`  
  Verwendung: UI (Fallback/Varianten, siehe `pubspec.yaml`)

> Hinweis: Die Schriften werden unverändert gebündelt; Reserved Font Names werden respektiert (keine Umbenennung).
